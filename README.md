## 📦 Installation

### Prerequisites
- Node.js 18+ 
- npm or yarn package manager

### Setup Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Vaishnavi09099/Vaishnavi_Portfolio_Website.git
   cd Vaishnavi_Portfolio_Website
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Set up environment variables** (if needed)
   ```bash
   cp .env.example .env.local
   # Update .env.local with your configuration
   ```

4. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

5. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🎨 Customization

### Colors & Theming
Edit `tailwind.config.ts` to customize your color scheme and theme:
```typescript
theme: {
  colors: {
    primary: '#your-color',
    secondary: '#your-color',
    // ...
  }
}
```

### Content
Update component files in the `components/` directory to add your:
- Project descriptions and links
- Skills and technologies
- Work experience
- Personal information
